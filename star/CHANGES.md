# 두더지 잡기 - 같은 사진 반복/위치 편중 수정 (3차)

## 이번 변경
- 같은 프로필 사진이 연속으로 다시 나오지 않도록 강화
  - `_MW_RECENT_AVOID` 2 → 3 (최근 3명까지 후순위로 밀림)
  - `_mwPickTarget`: 직전 1명 비교(8회 재시도) → **최근 N명 전체 제외** 방식으로 교체
  - 후보가 비면 전체 풀에서 안전하게 fallback
- 스폰 위치 랜덤 강화
  - 직전 스폰 위치를 `lastSpawnPos`로 기록
  - 가능한 경우(빈 칸이 2개 이상) 직전 위치와 다른 칸에서 등장
  - 새 게임 시작 시 `recentNames`/`lastSpawnPos` 초기화
- 캐시 무효화: `js/mole-whack-game.js?v=20260713-01` → `?v=20260713-02`

## 포함 파일
- js/mole-whack-game.js (반복방지 + 위치 랜덤 로직 보강)
