# 스타대학 데이터 센터 스모크 테스트 체크리스트

> 리팩터링 이후 핵심 기능이 깨지지 않았는지 빠르게 점검하기 위한 체크리스트
> 최종 업데이트: 2026-05-01

---

## 사용 방법

- 가능한 한 새로고침 후 시작
- 로컬 저장 데이터가 있는 상태와 없는 상태를 모두 한 번씩 보는 것이 좋음
- 각 항목은 “정상 표시”, “콘솔 에러 없음”, “클릭 후 다음 단계로 진행됨”을 기준으로 체크

---

## 1. 기본 로드

- [ ] 첫 진입 시 페이지가 정상 로드된다
- [ ] 콘솔에 치명적 에러가 없다
- [ ] 상단 탭이 모두 보인다
- [ ] 검색창이 보이고 입력 가능하다
- [ ] 다크 모드 전환이 동작한다
- [ ] 로그인 버튼 또는 로그아웃 버튼이 정상 표시된다

---

## 2. 메인 탭 진입

### 스트리머

- [ ] `스트리머` 탭 진입 가능
- [ ] 선수 목록이 렌더된다
- [ ] 종족 필터 버튼이 동작한다
- [ ] 검색이 동작한다
- [ ] `전적없음 숨김` 토글이 동작한다

### 현황판

- [ ] `현황판` 탭 진입 가능
- [ ] 카드 목록이 깨지지 않는다

### 티어 순위표

- [ ] `티어 순위표` 탭 진입 가능
- [ ] 정렬/필터가 동작한다

### 대전 기록

- [ ] `대전 기록` 탭 진입 가능
- [ ] 연도/월 필터가 보인다
- [ ] `0430년` 같은 잘못된 연도 옵션이 없다

### 개인전

- [ ] `개인전` 탭 진입 가능
- [ ] 입력/순위/기록 서브탭이 정상 동작한다

### 대학대전

- [ ] `대학대전` 탭 진입 가능
- [ ] 입력/순위/기록 서브탭이 정상 동작한다
- [ ] 연도 옵션이 정상이다

### 대회

- [ ] `대회` 탭 진입 가능
- [ ] 목록 또는 관련 화면이 깨지지 않는다

### 프로리그

- [ ] `프로리그` 탭 진입 가능

### 통계

- [ ] `통계` 탭 진입 시 지연 로딩 후 정상 표시된다

### 캘린더

- [ ] `캘린더` 탭 진입 시 지연 로딩 후 정상 표시된다

### 룰렛

- [ ] `룰렛` 탭 진입 시 지연 로딩 후 정상 표시된다

---

## 3. 선수 상세 모달

- [ ] 스트리머 목록에서 선수 클릭 시 상세 모달이 열린다
- [ ] 상단 헤더 카드가 정상 표시된다
- [ ] 대학/종족/티어 정보가 보인다
- [ ] 방송 채널 버튼이 있으면 정상 렌더된다
- [ ] `연도` 필터가 보인다
- [ ] `ELO 변화 추이`가 조건 충족 시 표시된다
- [ ] `모드별 전적` 섹션이 보인다
- [ ] `상대 종족별 전적` 섹션이 보인다
- [ ] `상대 전적` 섹션이 보인다
- [ ] `최근 경기 기록` 섹션이 보인다
- [ ] `종목` 필터가 보인다
- [ ] 시즌 필터가 있는 경우 동작한다
- [ ] 페이지네이션이 동작한다
- [ ] 같은 대학 팀원 섹션이 보인다
- [ ] 관리자 로그인 시 메모 섹션이 보인다

---

## 4. 선수 상세 상호작용

- [ ] 연도 필터 변경 시 상세 내용이 재빌드된다
- [ ] 종목 필터 변경 시 최근 경기 기록이 갱신된다
- [ ] 시즌 필터 변경 시 최근 경기 기록이 갱신된다
- [ ] 상대 전적 정렬이 동작한다
- [ ] 공유 카드 버튼이 동작한다
- [ ] 이미지 저장 버튼이 동작한다
- [ ] 관리자 상태면 수정 버튼이 동작한다

---

## 5. 대학 상세 모달

- [ ] 대학명 클릭 시 대학 상세 모달이 열린다
- [ ] 대학 헤더 카드가 정상 표시된다
- [ ] 소속 스트리머 목록이 보인다
- [ ] 상대 대학 전적 섹션이 보인다
- [ ] 최근 대전 기록 섹션이 보인다
- [ ] 에이스 카드 섹션이 보인다
- [ ] 모달 내 필터/정렬이 있는 경우 정상 동작한다
- [ ] 공유 카드 버튼이 동작한다
- [ ] 이미지 저장 버튼이 동작한다

---

## 6. 연도 관련 회귀 테스트

- [ ] 모든 연도 드롭다운에서 `0430년` 같은 잘못된 값이 없다
- [ ] 실제 데이터가 있는 연도만 표시된다
- [ ] 연도 정렬이 오름차순으로 맞다

점검 권장 위치:

- [ ] 대전 기록 탭
- [ ] 대학대전 기록/입력 관련 화면
- [ ] 선수 상세 연도 필터

---

## 7. 드래그 스크롤 회귀 테스트

- [ ] 상단 탭 바를 마우스 드래그로 좌우 스크롤할 수 있다
- [ ] 필터 바를 마우스 드래그로 좌우 스크롤할 수 있다
- [ ] 기록 인라인 필터 바를 드래그할 수 있다
- [ ] 드래그 후 버튼 오클릭이 과하게 발생하지 않는다
- [ ] 버튼/인풋 위에서는 드래그 시작이 비정상적으로 먹지 않는다

---

## 8. 검색/필터 기본 점검

- [ ] 메인 검색 입력 후 렌더가 깨지지 않는다
- [ ] 필터 변경 후 포커스가 과하게 튀지 않는다
- [ ] 검색 상태에서 렌더 후 포커스 복원이 지나치게 어색하지 않다

---

## 9. 공유/저장 기능

- [ ] 공유 카드 열기 가능
- [ ] 공유 카드에서 닫기 가능
- [ ] 이미지 저장 버튼 동작
- [ ] 캡처 후 콘솔 에러 없음

---

## 10. 관리자 기능

관리자 로그인 상태에서 점검:

- [ ] 선수 수정 진입 가능
- [ ] 기록 수정/삭제 버튼 노출 정상
- [ ] 선택 삭제 또는 일괄 액션이 정상 동작
- [ ] 저장 후 렌더가 정상 복구

---

## 11. 콘솔 확인 항목

각 테스트 중 함께 확인:

- [ ] `ReferenceError` 없음
- [ ] `buildPlayerDetailHTML is not defined` 없음
- [ ] `buildUnivDetailHTML is not defined` 없음
- [ ] `render is not defined` 없음
- [ ] `enableDragScroll is not defined` 없음
- [ ] `isValidAppYear is not defined` 없음
